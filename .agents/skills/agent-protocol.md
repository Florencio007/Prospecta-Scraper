# Skill : Agent Maintenance Protocol

Ce skill définit la manière dont l'IA doit interagir avec le projet pour garantir la pérennité des connaissances.

## 🏁 AVANT toute modification
1. **Consulter les Skills** : Lister et lire les fichiers dans `.agents/skills/` qui sont liés au domaine d'intervention (ex: `playwright-scraper.md` pour un changement de script).
2. **Vérifier la cohérence** : S'assurer que la modification prévue respecte les standards établis.

## 🚀 PENDANT la modification
1. Appliquer les changements techniques.
2. Garder une trace des décisions architecturales.

## 🏁 APRÈS toute modification
1. **Mise à jour des Skills** : Si la modification introduit un nouveau pattern, une nouvelle colonne DB, ou un nouveau composant, mettre à jour immédiatement le skill correspondant.
2. **Documentation** : Si aucun skill n'existe pour le nouveau domaine, en créer un.

> [!IMPORTANT]
> Ce processus est **OBLIGATOIRE** et doit être effectué de manière autonome sans demande explicite de l'utilisateur.
