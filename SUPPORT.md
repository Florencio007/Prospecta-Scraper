## 🐛 Problèmes Courants et Solutions

### Authentification

**❌ "Invalid email"**
→ Vérifier le format de l'email

**❌ "Password too short"**
→ Mot de passe doit faire minimum 8 caractères

**❌ "User already exists"**
→ Email déjà utilisé, utiliser "Mot de passe oublié"

### Données

**❌ "No prospects found"**
→ Vérifier que des prospects ont été ajoutés

**❌ "Export failed"**
→ Essayer un autre format (CSV à la place de XLSX)

**❌ "API error 401"**
→ Vérifier les variables d'environnement Supabase

### Performance

**❌ "Page is slow"**
→ Vider le cache du navigateur
→ Vérifier la connexion internet
→ Réduire le nombre de filtres

### Mobile

**❌ "Layout broken on mobile"**
→ Forcer le portrait mode
→ Zoom à 100%
→ Vider le cache du navigateur

## 📋 FAQ

### Général

**Q: Prospecta est-il gratuit?**
A: Oui, le plan Starter est gratuit avec 50 prospects max. Plan Pro: 49,000 Ar/mois

**Q: Quels navigateurs sont supportés?**
A: Chrome, Firefox, Safari, Edge (dernières versions)

**Q: Mes données sont-elles sécurisées?**
A: Oui, nous utilisons HTTPS, encryption, et Row Level Security

**Q: Puis-je exporter mes données?**
A: Oui, en multiple formats (CSV, XLSX, JSON, PDF)

### Utilisation

**Q: Comment ajouter un prospect?**
A: Aller à Prospects → Nouveau prospect → Remplir le formulaire

**Q: Comment créer une campagne?**
A: Aller à Campagnes → Nouvelle campagne → Définir les critères

**Q: Puis-je avoir plusieurs utilisateurs?**
A: Plan Starter: 1 seul. Plan Pro: À demander. Plan Enterprise: Illimité

**Q: Comment intégrer Google, Facebook, etc?**
A: Feature en développement pour Phase 2 (Q2 2026)

### Technique

**Q: Puis-je auto-héberger Prospecta?**
A: Oui, code open sur GitHub. Voir [DEPLOYMENT.md](./DEPLOYMENT.md)

**Q: Quelle est l'API?**
A: API REST en développement pour Phase 2

**Q: Puis-je créer des extensions?**
A: Oui, voir [CONTRIBUTING.md](./CONTRIBUTING.md)
